# Software Project Management Database

This project contains a fully normalised relational database (3NF) built using MySQL and phpMyAdmin.

## Features

- Client and project management
- Pool member skill tracking with experience levels
- Project skill requirements matching
- Automated candidate ranking for project allocation
- Skill gap analysis

## Files

- R1_schema_and_data.sql → Database structure and sample data
- R2_queries.sql → Queries for testing, reporting, and decision-making

## Technologies

- MySQL (XAMPP)
- phpMyAdmin
